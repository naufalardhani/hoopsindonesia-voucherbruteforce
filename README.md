## How to Install

```bash
> git clone https://github.com/naufalardhani/hoopsindonesia-voucherbruteforce.git
> cd hoopsindonesia-voucherbruteforce
```

## How to Usage

```bash
> nano config.yaml
    [edit your token]
    [edit start and finish id (integer)]
> python3 brute.py
```
